# LTRROE: Фреймворк стохастического моделирования проектных рисков

## Исследовательский проект

**LTRROE** (Local Team Resource & Risk Optimization Engine) — независимый исследовательский проект, посвящённый стохастическому моделированию проектных рисков. Система демонстрирует применение теории вероятностей, исследования операций и вычислительных методов к задачам управления проектами.

### Научный контекст
Работа адресует критическую потребность в вероятностных подходах к планированию проектов, выходя за рамки детерминистических моделей для учёта неопределённости в длительностях задач, доступности ресурсов и человеческих факторов.

### Теоретические основы
Проект основывается на установленных методологиях:
- **PERT/CPM** - Метод критического пути с вероятностными расширениями
- **Моделирование Монте-Карло** - Стохастическая квантификация неопределённости
- **Исследование операций** - Оптимизация ресурсов при ограничениях
- **Инженерия человеческих факторов** - Моделирование производительности на основе навыков

### Математическая модель
Проекты формализуются как ориентированные ациклические графы (DAG) $G = (V, E)$, где:
- Вершины $v_i \in V$ представляют задачи с треугольными распределениями длительности $T_i \sim \text{Triangular}(a_i, m_i, b_i)$
- Рёбра $e_{ij} \in E$ представляют ограничения предшествования
- Каждая задача имеет связанные требования к навыкам и ограничения ресурсов

### Методологические вклады
1. **Интегрированный стохастическо-детерминистический фреймворк**: Комбинирует CPM с моделированием Монте-Карло
2. **Квантификация человеческих факторов**: Моделирует эффективность сотрудников как зависящие от навыков случайные величины
3. **Многомерные метрики риска**: Вычисляет распределения вероятностей для сроков завершения, утилизации ресурсов и разрывов в навыках
4. **Конвейер визуальной аналитики**: Преобразует вычислительные результаты в действенные инсайты

### Реализация
Система реализована на Python с модульной архитектурой:
- **Вычислительное ядро** - Алгоритмы стохастического моделирования
- **Модели данных** - Типизированные сущности проекта
- **Слой визуализации** - Аналитические дашборды
- **Модуль валидации** - Проверки целостности данных
