# LTRROE: Фреймворк стохастического моделирования проектных рисков

> **Описание академического исследовательского проекта**

## Направление исследования

**LTRROE** (Local Team Resource & Risk Optimization Engine) — независимый исследовательский проект, изучающий стохастические подходы к оценке проектных рисков. Система формализует проектное планирование как вероятностную задачу оптимизации, интегрируя методы исследования операций, теории вероятностей и вычислительного моделирования.

Фреймворк спроектирован как **baseline** для последующих подходов **машинного обучения** к предсказанию проектных рисков.

---

## Теоретическая основа

Проект базируется на установленных методологиях:

1.  **Теория вероятностей:** Треугольные распределения (PERT), симуляция Монте-Карло.
2.  **Исследование операций:** Метод критического пути (CPM), распределение ресурсов.
3.  **Теория графов:** Ориентированные ациклические графы (DAG) для моделирования зависимостей.
4.  **Инженерия человеческих факторов:** Количественная оценка производительности на основе навыков.

### Математическая формализация

Проекты представлены как ориентированные ациклические графы $G = (V, E)$:

*   **Вершины** $v_i \in V$: Задачи с треугольным распределением длительности $T_i \sim \text{Triangular}(a_i, m_i, b_i)$.
*   **Рёбра** $e_{ij} \in E$: Ограничения предшествования с опциональными временными лагами.
*   **Дополнительные ограничения:** Доступность ресурсов и квалификация исполнителей.

## Методологический вклад

### 1. Интегрированный стохастически-детерминистический анализ
*   Комбинирует детерминистический **CPM** с вероятностной симуляцией **Монте-Карло**.
*   Вычисляет распределения вероятностей для дат завершения.
*   Идентифицирует критические пути в условиях неопределённости.

### 2. Количественная оценка человеческого фактора
*   Эффективность сотрудников моделируется как случайная величина, зависящая от навыков.
*   Факторы замедления производительности на основе **skill gaps** (разрывов в навыках) и нагрузки.
*   Реалистичное моделирование кривых обучения и эффектов усталости.

### 3. Многомерные метрики риска
*   Вероятность завершения для произвольных дедлайнов.
*   Распределения загрузки ресурсов (Heatmaps).
*   Анализ дефицита навыков и рекомендации по составу команды.

### 4. Конвейер визуальной аналитики
*   Автоматическая генерация аналитических дашбордов.
*   Интерактивное исследование результатов симуляций.
*   Сравнительный анализ различных сценариев.

### 5. Интеграция машинного обучения (фаза разработки)
*   Регрессионные модели для автоматической оценки параметров задач.
*   Анализ исторических данных проектов для подбора распределений.
*   Сравнительная оценка: **экспертные оценки vs ML-предсказания**.
*   Feature engineering характеристик проектов.

*   **Слой визуализации:** Многоформатная аналитическая отчётность.

## Академическая значимость

Данный проект демонстрирует:
1.  Применение стохастических методов к практическим инженерным задачам.
2.  Интеграцию человеческих факторов в вычислительные модели.
3.  Разработку масштабируемых фреймворков для анализа сложных систем.
4.  Подготовку к продвинутым исследованиям в исследовании операций и оптимизации.
5.  Создание **baseline** для ML-подходов к предсказанию проектных рисков.
