# LTRROE v2.0 — Local Team Resource & Risk Optimization Engine

**Система интеллектуальной поддержки принятия решений для управления проектными рисками с использованием вероятностных моделей и визуализации**

---

## Аннотация

Проект представляет собой учебно-исследовательскую разработку в области **интеллектуального анализа данных** и **исследования операций**. Система решает задачу моделирования сложных проектных систем, оценки неопределенности сроков выполнения (метод Монте-Карло) и оптимизации распределения ресурсов с учетом человеческого фактора (квалификация, утомляемость).

Разработка соответствует профилю кафедры, интегрируя методы **машинного обучения** (на стадии разработки), **вероятностного моделирования**, **экспертных систем** и **интерактивной визуализации данных** для поддержки принятия управленческих решений.

---

## Архитектура системы

Проект реализован на языке Python и имеет модульную структуру:

```
project/
├── models.py              # Объектная модель предметной области
├── test_data.py           # Готовый пласт данных для системы
├── algorithmes.py         # Вычислительное ядро: стохастическое моделирование и оптимизация
└── visualisation.py       # Модуль визуализации данных и аналитики (графики, диаграммы)
```

## Возможности системы

### 1. Вероятностное планирование проекта
- **Forward/Backward Pass** - классический метод критического пути (CPM)
- **Учет зависимостей задач** - сетевой график с предшественниками/последователями
- **Расчет резервов времени** - автоматическое определение slack для каждой задачи

### 2. Стохастическое моделирование рисков
- **Метод Монте-Карло** (1000+ итераций) для оценки вероятности завершения
- **Треугольное распределение** (PERT) для моделирования неопределенности длительностей
- **Учет человеческого фактора**:
  - Коэффициент медлительности на основе навыков
  - Учет перегрузки сотрудников
  - Анализ соответствия компетенций

### 3. Анализ ресурсов и компетенций
- **Матрица компетенций** - анализ соответствия навыков сотрудников требованиям задач
- **Тепловые карты загрузки** - визуализация распределения нагрузки по дням
- **Выявление перегрузок** - автоматическое обнаружение сотрудников с превышением нормы

### 4. Расширенная визуализация результатов
- **Диаграмма Ганта** с цветовой кодировкой по критичности задач
- **Гистограммы распределения** результатов симуляции Монте-Карло
- **Радар-диаграммы** анализа разрывов компетенций команды
- **Heatmap загрузки** сотрудников с выделением проблемных зон

2. **Интеллектуализация системы:**
   - Внедрение машинного обучения для прогнозирования коэффициентов медлительности
   - Использование генетических алгоритмов для автоматической оптимизации назначений

3. **Практическое применение:**
   - Интеграция с системами управления проектами (Jira, Trello)
   - Разработка веб-интерфейса для менеджеров проектов

4. **Исследовательские направления:**
   - Анализ влияния психологических факторов на продуктивность
   - Моделирование эффектов обучения и накопления опыта

## Соответствие научным направлениям кафедры

Проект развивает компетенции в рамках ключевых дисциплин:

- **Вероятностные модели в ML:** Применение метода Монте-Карло, треугольных распределений
- **Интеллектуальные экспертные системы:** Автоматическое выявление bottlenecks, рекомендации
- **Исследование операций:** Оптимизация расписания, балансировка нагрузки
